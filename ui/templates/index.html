python bank with some cool features

Simple banking website created to learn flask to provide users with web banking experience, user authentification to ensure the security of the accounts, nosql database in order to save all of the user accounts.

Transactions
Send and receive money from other users, search them by email address or mobile number. Request money from users who are planning on sending you money later on.
Savings accounts
Unlike other banks, we offer users with daily interest payments for their money, starting from 0.01% every day. Lets make some good money together!
Loans
We have partnered with local mafia to lend you some money during the uncertain times. Don't worry about background-scans, reasoning, we do not ask any questions.
Gambling
We heard you, and created our own online casino, where you can use your own funds directly from our bank, as well as get a loan when you run dry.
